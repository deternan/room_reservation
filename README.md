# room_reservation

## How to start web application?
```
// Build DB
$ python manage.py migrate
// Open http://127.0.0.1:8000/reservation
$ python manage.py runserver
```

## Pre Requirement
`$ pip install -r requirements.txt`

* Login - reservation/
* Index - reservation/index
